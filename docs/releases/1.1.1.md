# gTunes 1.1.1

gTunes 1.1.1 is a keyboard navigation and search-flow maintenance release for the Linux Jellyfin music client.

## Highlights

- Let Enter immediately open the active invisible-search match on album and artist collection pages.
- Let Enter open the focused album after keyboard navigation inside an artist's album grid.
- Expand Escape into a universal navigation key: clear and leave search, back out of nested collection views, or return the active list/grid to the top.
- Fix a search-entry clearing crash caused by GTK search-change signal reentrancy.
