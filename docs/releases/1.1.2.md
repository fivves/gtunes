# gTunes 1.1.2

gTunes 1.1.2 is The Radio Release for the Linux Jellyfin music client.

## New Features

- Add a dedicated Radio page for local internet radio stations.
- Add built-in stream presets and support for custom direct stream URLs.
- Add YouTube Live and Twitch Live station support, resolved to audio-only streams at playback time.
- Keep radio playback integrated with the existing player controls, now playing state, and page navigation.

## Fixes/Refinements

- Refine radio stream handling and resume behavior.
- Force radio playback to request audio-only streams.
- Fix responsive radio grid column behavior.
- Improve radio station card layout, remove-button placement, and station text casing.
- Fix now playing navigation links and the active track indicator.
- Preserve collection scroll position and highlighted items when navigating back.
- Match sidebar cover artwork resolution.
