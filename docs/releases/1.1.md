# gTunes 1.1

gTunes 1.1 continues the 1.0 Linux Jellyfin music client release line with a focus on keeping now playing state stable while browsing and refreshing the library.

## Highlights

- Ship a release AppImage that runs as a standalone download without requiring Rust, Cargo, or the development headers used to build gTunes.
- Preserve the now playing selection when the library refreshes.
- Preserve the now playing album selection when navigating between views.
- Decouple the playback queue from browsing so queue changes do not disturb the current browsing context.
- Fix CI formatting and Clippy warnings from the release branch.
