# gTunes 1.1.4

gTunes 1.1.4 is The Stability Update for the Linux Jellyfin music client.

This is not a big feature release. It focuses on making the core of the app more robust.

## Highlights

- Move more playback, queue ordering, radio mode, gapless handoff, and restore state into the playback session so state transitions stay centralized and predictable.
- Preserve the current playback position during transcode fallback so stream recovery is less disruptive when a direct stream has to be retried.
- Add targeted playback session test coverage around restore ordering, queue helpers, and fallback behavior.

## Fixes and Refinements

- Decouple Discord artwork uploads and presence updates from UI cache and MPRIS wiring to reduce cross-system playback side effects.
- Add Nerd Font icon support for radio stations so the radio page can show cleaner custom station markers.
- Add `Ctrl+S` as a direct shuffle toggle and refresh the keyboard shortcuts menu and related documentation.
- Simplify the app window title and refine keyboard shortcut support, including more consistent playback behavior around queue cleanup and radio exit.
- Update release-facing documentation and screenshots to match the current app.
